🔑 Groq API Key
Your API key is stored only in your browser's localStorage — it is never sent to any server except Groq's API directly from your browser. It is never embedded in the source code.
How to get a free Groq API key:
1. Go to console.groq.com
2. Sign up (free, no credit card needed)
3. Go to API Keys → Create API Key
4. Copy & paste it above
5. Optional: In Groq console → Data Controls → Enable Zero Data Retention

🔒 Data & Privacy
✓ API Key — Stored only in your browser's localStorage. Never in code, never on any server.
✓ Portfolio Data — Stored only in your browser's localStorage. Never leaves your device.
✓ Stock Queries — Sent directly to Yahoo Finance (free, public API) & Groq for AI. No middleman.
✓ Groq Privacy — Groq is contractually prohibited from using your data for model training by default.
ℹ About StockIQ
StockIQ is a personal stock analysis tool using:

• Yahoo Finance — Free real-time stock data (US & India)
• Groq API — Free AI analysis (Llama, Qwen models)
• Chart.js — Price history charts

This is a static website — no backend, no server, no database. Everything runs in your browser.

Not financial advice. Always do your own research.
